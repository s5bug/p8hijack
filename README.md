# p8hijack

```
Usage: p8hijack [--image <input.png>] <https://input.url> <output.p8.png>

PICO-8 Web Hijacker

Options and flags:
    --help
        Display this help text.
    --image <input.png>, -i <input.png>
        Set the cover image to be used for steganography.
```

```
p8hijack https://exok.com/minigames/celeste.html celeste.p8.png
```
